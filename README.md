# SNAPFOREST E-Commerce

Production-ready e-commerce platform built with Next.js, TypeScript, and Tailwind CSS.

## Features
- Product browsing with category filters
- Shopping cart functionality
- Responsive design
- Modern UI matching the provided design

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide React Icons
